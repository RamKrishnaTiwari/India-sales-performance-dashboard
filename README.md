#India Sales Performance Dashboard

📌 Overview

This project features a Power BI dashboard focused on the Region and State-wise distribution of Sales in India. It provides a comprehensive view of retail or B2B performance across the country, allowing stakeholders to evaluate top-performing states, regional market shares, and track current sales metrics against target quantities.

📊 Key Performance Indicators (KPIs)

The dashboard tracks dynamic, high-level metrics on the right-hand panel:
•	Total Sales by Region: Displays dynamic total sales based on user selection (e.g., highlighting 743,038 for the West region).
•	Average Quantity Target (Gauge Chart): Tracks the current average quantity (currently at 35) against a defined target range, providing a quick visual on performance health.

📈 Visualizations & Insights

The dashboard includes several detailed visualizations to break down the geographic sales data:
•	Total Sales by State (Column Chart): Ranks states by total sales volume, revealing that Delhi and Maharashtra are the leading markets, followed by Tamil Nadu, Bihar, and Uttar Pradesh.
•	Total Sales and Quantity by State (Line/Combo Charts): Two separate visual variations that map the correlation between total sales revenue and the quantity of items sold across various states like Kerala, Punjab, Gujarat, and West Bengal.
•	Sales Share by State for each Region (Donut Chart): Provides a percentage breakdown of sales distribution, color-coded by macro-regions (North, South, West, and East) to show how individual states contribute to their larger regional territories.

🎛️ Interactive Filters (Slicers)

To allow for highly targeted analysis, the dashboard includes dynamic filtering options located at the top right:
•	Date Range Slider: Enables users to filter metrics for a specific timeframe between 09-01-2021 and 01-01-2024.
•	Region Slicers: Quick-select buttons to isolate data specifically for the East, North, South, or West regions.

🛠️ Tools Used

•	Power BI: For data visualization, geographic performance tracking, and interactive dashboard design.

🚀 How to Use

1.	Clone this repository: git clone https://github.com/your-username/india-sales-dashboard.git
2.	Open the .pbix file using Microsoft Power BI Desktop.
3.	Use the date slider and regional buttons on the right-hand panel to interact with the visualizations and uncover specific insights for targeted timeframes or geographic territories.
